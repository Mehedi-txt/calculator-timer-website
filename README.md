# Calculator + Timer Website

A simple, clean, and beginner-friendly **Calculator + Timer web application** built using HTML, CSS, and JavaScript.  
This project combines two useful tools in a single responsive webpage.

Live Demo: https://mehedi-txt.github.io/calculator-timer-website/

---

## Features

### Calculator
- Basic arithmetic operations (+, -, ×, ÷)
- Clean and simple UI
- Easy-to-use button interface
- Instant result calculation

### Timer
- Start, pause, and reset functionality
- Countdown timer system
- Simple and distraction-free design
- Works smoothly in browser

---

## Built With

- HTML5 – Structure
- CSS3 – Styling and layout
- JavaScript – Logic and functionality
